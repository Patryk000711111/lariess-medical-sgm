# L'Ariess Medical SGM - Web Preview

To jest statyczna wersja podglądowa systemu webowego L'Ariess Medical SGM.

## Zawartość
- `index.html` — ekran główny aplikacji
- `styles.css` — stylowanie interfejsu
- `script.js` — logika frontendowa
- `data.json` — dane wejściowe (skrzynek i punktów)

## Uruchomienie lokalne
Wystarczy otworzyć `index.html` w przeglądarce.

## Wgranie na GitHub
1. Utwórz nowe repozytorium na GitHub.
2. Wgraj wszystkie pliki z tego folderu.
3. Jeśli chcesz wdrożyć na Vercel, połącz repozytorium z Vercel i uruchom deploy.
