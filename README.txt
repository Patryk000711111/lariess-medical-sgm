# L’Ariess Medical SGM — Web Preview

To jest gotowa paczka statycznego web preview.
## Jak uruchomić lokalnie
1. Rozpakuj plik ZIP.
2. Otwórz plik `index.html` w przeglądarce.

## Jak wrzucić na Vercel
1. Zaloguj się do Vercel.
2. Na stronie głównej wybierz import / upload projektu.
3. Wskaż rozpakowany folder z tymi plikami.
4. Opublikuj projekt.
